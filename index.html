Quick help

This first build supports standard NES .nes files. Use your own ROM files only. No commercial games or BIOS files are included.

Controls: D Pad, A, B, Select and Start. Keyboard fallback is Arrow keys, X for A, Z for B, Enter for Start, Right Ctrl for Select.

Core: JSNES, loaded from unpkg CDN. If the page is offline, the emulator core will not load in this v0.1 build.
