TonyNES Web App v0.1

Upload index.html and manifest.webmanifest to GitHub Pages.

Open the page on iPhone, then use Safari, Share, Add to Home Screen.

This v0.1:
- loads local .nes files from the iPhone file picker
- uses JSNES from unpkg CDN
- has a portrait handheld layout
- has D Pad, A, B, Select, Start and Reset buttons
- includes no ROMs, commercial games, Nintendo branding or BIOS files

Notes:
- NES only in this first build.
- It needs internet access to load the JSNES core from the CDN.
